# X Project - Interactive Quote Tool

Interactive quote calculator for UI/UX design services.

## Features
- Selectable design phases
- Landing page sections pricing
- UI Kit selection
- Optional add-ons
- Additional screens counter
- Live price calculation
- CSV export
- Email submission

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Deployment
Deployed on Vercel.
